ğŸ“‚ Introduction
Folder Dissolver is an enhanced productivity tool designed specifically for Windows users. It allows you to extract all contents of a folder to its parent directory and automatically delete the original folder with just one click.

âœ¨ Key Features
Multi-language Support: Automatically adapts the management interface and right-click menu.

Perfect Restore: Records the original directory structure for one-click restoration via the Management Center.

Recursive Toggle: Press R during the countdown to toggle between "Top Level Only" and "Full Recursive" mode.

Safety Mechanism: 5-second high-frequency response countdown, supporting Enter to skip and Ctrl+C to cancel.

Duplicate Protection: Automatically detects filename conflicts and appends a timestamp to prevent data loss.

One-click Install/Uninstall: Clean setup and removal without registry residue.

Right-click the file and select "Run with PowerShell".

Follow the on-screen prompts to select your language and complete the installation.

Right-click a folder to "Dissolve", or right-click the desktop background to open the "Manager" for restoration.
