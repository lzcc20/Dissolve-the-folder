## ğŸ“‚ Introduction
**Dissolve-the-folder** is an enhanced productivity tool designed specifically for Windows users. It allows you to extract all contents of a folder to its parent directory and automatically delete the original folder with just one click.

## âœ¨ Key Features
- **Multi-language Support**: Automatically adapts the management interface and right-click menu based on your choice.
- **One-click Install/Uninstall**: An integrated management script ensures a clean setup and removal without residue.
- **Safety Mechanism**: Features a 5-second confirmation countdown, supporting **Enter** to skip and **Ctrl+C** to cancel instantly.
- **Duplicate Protection**: Automatically detects filename conflicts and appends a timestamp to prevent data loss.
